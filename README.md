# HERO Ops — Live Demo

**HERO Aviation, LLC** · Operational Safety Systems

**▶ Open the live demo: https://heroaviationllc.github.io/hero-ops-demo/**

A fully interactive demo of the HERO Ops safety-management platform, seeded
with a fictional operator (Canyon County Air Rescue). No real data. Nothing
you do in the demo is saved — refresh and it resets.

## Viewing it the first time

1. **Open the link above** on any device — iPhone, iPad, Android, Mac, or
   Windows. Nothing to install; it runs in the browser.
2. A short **welcome guide appears on first load** walking you through the
   demo (it's the same three steps below). Reopen it anytime with the **?**
   button in the corner.
3. **Sign in as any crew member.** Pick **Joel Smith** (Chief Pilot /
   manager) or **K. Whitfield** (Safety Officer) to see every feature —
   risk matrix, Safety Health analytics, personnel & fleet management.
4. **Everything is clickable.** File a FRAT and watch it auto-score, log a
   flight, file a hazard report with photos, run a preflight, check
   currency, condition the hoist.

## Make it feel like an app (phones & tablets)

- **iPhone / iPad (Safari):** Share → **Add to Home Screen**
- **Android (Chrome):** ⋮ menu → **Add to Home screen**

It then launches full-screen from the home screen with the HERO dark theme.

## What this demo represents

In production, every operator receives their **own isolated deployment and
database** — flight logs, FRATs, hazard reports, biometric readiness, and
currency tracking in one system, with your data never sharing a system with
anyone else's.
